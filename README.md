# 🛠 Paramount Enterprises – Hardware Shop Management System

A full-stack MERN (MongoDB, Express.js, React.js, Node.js) web application developed for Paramount Enterprises, a hardware shop business. The system streamlines inventory management, customer management, employee administration, order processing, supplier management, reporting, and online shopping through a centralized platform.

## 📌 Project Overview

The application consists of two main modules:

### 🔧 Admin Panel
Designed for staff and management to handle day-to-day business operations.

#### Features
- Role-based authentication (Admin / Staff)
- Employee Management
- Inventory & Stock Management
- Customer Management
- Supplier Management
- Order Processing
- Analytics Dashboard
- Low Stock Alerts
- PDF Report Generation

---

### 🛒 Customer Portal
An e-commerce platform that allows customers to browse and purchase products online.

#### Features
- User Registration & Login
- Product Browsing
- Search & Filtering
- Shopping Cart
- Checkout Process
- Online Payment Integration
- Order Tracking
- Account Management

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/nilavannan/paramount-enterprises.git
```

### Install Dependencies

```bash
npm install
```

### Run Application

Admin Panel:

```bash
cd admin-panel
npm install
npm start
```

Customer Portal:

```bash
cd customer-portal
npm install
npm start
```

## 🧪 Testing

Run tests:
